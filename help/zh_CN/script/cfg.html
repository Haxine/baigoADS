配置

配置文件为一个 JSON 文件，必须命名为 config.json，系统将读取此文件，以便使脚本生效。

键名 类型 说明 备注
name string 脚本名 创建（编辑）广告位时，选择脚本，会将此参数填入广告位名称。
plugin string 插件名 创建（编辑）广告位时，选择脚本，会将此参数填入插件名。此处指广告脚本的 jQuery 插件名。
selector string 默认选择器 创建（编辑）广告位时，选择脚本，会将此参数填入默认选择器。
isPercent string 是否允许按几率展现 enable 或 disable，创建（编辑）广告位时，选择脚本，会自动选择允许按几率展现。
count int 显示广告数 每个广告位上允许同时显示的广告数，如：Banner、飘动、卷帘等一般为 1 个，轮播可以为数个，可根据实际情况决定。自 2.0 起
type string 广告内容 attach 或 text，创建（编辑）广告位时，选择脚本，会自动选择广告内容。
note string 备注 创建（编辑）广告位时，选择脚本，会自动显示本参数。
opts array 选项，多维数组 查看 opts 对象。
 

opts 对象

选择脚本，会自动在表单中添加选项表单，并在生成广告代码时，成为广告插件的 JSON 对象。

键名 类型 说明 备注
field string 选项名 表单子健，在生成广告代码时，此对象为选项键名。
label string 说明文字 表单说明，在生成广告代码时，此对象为备注。
value string 默认值 填入表单，在生成广告代码时，此对象为选项键值。
 

配置文件示例

重要提示：由于系统将读取 JSON 文件，并转换为 PHP 能够支持的数组，因此 JSON 文件中不能含有任何注释，否则系统将会无法正确读取。 

{
    "name": "Banner",
    "plugin": "adsBanner",
    "selector": "#adsBanner",
    "isPercent": "enable",
    "type": "attach",
    "note": "本脚本需要 jQuery 支持",
    "opts": [
        {
            "field": "loading",
            "label": "载入消息",
            "value": "正在载入广告 ..."
        }
    ]
}
