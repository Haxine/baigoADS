概述

请注意区分此处的广告脚本插件和插件和之间的区别，广告脚本插件一般指 JavaScript 开发的 jQuery 插件，属于客户端脚本。而插件是指系统级的，一般用 PHP 语言开发，属于服务器端程序。

本系统的内置的常用广告脚本均基于 jQuery 库，少数基于 bootstrap，推荐开发者使用 jQuery 开发。

广告脚本存放于 ./advert/ 目录，开发者开发的脚本请存放在此，一个脚本一个目录，所有目录、文件名必须使用英文名，目录下必须有三个文件 script.js 为脚本文件、config.json 为配置文件、style.css 为样式文件，推荐加入 readme.txt 文件，方便使用者。

开发者完全可以通过查看内置的常用广告脚本来进行开发参考。

目录结构说明

文件 用途 说明
config.json 配置文件 配置文件，JSON 格式。
script.js 脚本文件 脚本文件，Javascript 格式。
style.css 样式文件 样式文件，css 格式。
readme.txt 使用说明 使用说明，文本格式。
