使用

开发工作完成后，请将文件上传至 ./advert/ 目录，然后在管理后台创建（编辑）广告位，选择新上传的脚本，保存成功后，点击 “预览 / 获取代码”，将示例代码复制到需要显示广告的网页即可。

为了更灵活的运用，您也可以自行修改代码，以便达到您需要的目的，详情请看示例代码。

 

示例代码

<!DOCTYPE html>
<html lang="zh">
    <head>
        <title>advert</title>

        <!-- jQuery begin -->
        <script src="http://www.domain.com/static/js/jquery.min.js" type="text/javascript"></script>
        <!-- 系统内置 jQuery，地址一般为 /static/lib/jquery.min.js 也可另行载入 -->

        <!-- 广告样式 begin -->
        <link href="http://www.domain.com/advert/banner/style.css" type="text/css" rel="stylesheet">
        <!-- 此部分为可选项，请根据实际情况决定 -->

    </head>
    <body>

        <!-- 网页其他内容 -->
        ...


        <!-- 显示容器 begin -->
        <div id="adsBanner_1"></div>
        <!-- 此部分为必须项，ID 或者 CLASS 请与初始化时的选择器相对应，您还可以根据实际情况，添加样式控制 -->



        <!-- 网页其他内容 -->
        ...


        <!-- 初始化 begin -->
        <script type="text/javascript">
        $(document).ready(function(){
            _opts_ad_1 = { //定义选项
                loading: "正在载入广告 ...", //载入消息
                ..., //您还可以根据脚本自行添加选项
                data_url: "http://www.domain.com/advert/index.php?m=advert&c=request&a=list&posi_id=2" //广告数据，此为必须项
            }
            $("#adsBanner_1").adsBanner(_opts_ad_1); //初始化脚本对象，此处的选择器请与显示容器的 ID 或者 CLASS 相对应
        });
        </script>
        <!-- 此部分为必须项 -->

        <!-- 广告脚本 begin -->
        <script src="http://www.domain.com/advert/banner/script.js" type="text/javascript"></script>
        <!-- 此部分为必须项 -->

    </body>
</html>
